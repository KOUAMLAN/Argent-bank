# Projet Argent Bank

Ce projet contient le frontend et le backend complets pour l'application Argent Bank, une plateforme moderne pour la gestion des finances personnelles.

## Structure du projet

- `/`: Contient l'application frontend React construite avec Vite.
- `/backend`: Contient le serveur d'API Node.js/Express.

---

## Configuration du Backend

Le serveur backend fournit des points de terminaison d'API pour l'authentification des utilisateurs et la gestion de leur profil.

### Prérequis

- [Node.js](https://nodejs.org/) (version 16 ou ultérieure)
- npm (fourni avec Node.js)

### Installation & Lancement

1.  **Naviguez vers le répertoire backend :**
    ```bash
    cd backend
    ```

2.  **Installez les dépendances :**
    ```bash
    npm install
    ```

3.  **Démarrez le serveur :**
    ```bash
    npm start
    ```

Le serveur démarrera sur `http://localhost:3001`.
Vous pouvez consulter la documentation de l'API via Swagger à l'adresse `http://localhost:3001/api-docs`.

---

## Configuration du Frontend

Le frontend est une application monopage (SPA) construite avec React, Redux, et Tailwind CSS.

### Prérequis

- [Node.js](https://nodejs.org/) (version 16 ou ultérieure)
- npm (fourni avec Node.js)

### Installation & Lancement

1.  **Naviguez vers le répertoire racine du projet** (où se trouve ce README).

2.  **Installez les dépendances :**
    ```bash
    npm install
    ```

3.  **Démarrez le serveur de développement :**
    ```bash
    npm run dev
    ```

L'application s'ouvrira dans votre navigateur à l'adresse `http://localhost:5173` (ou un autre port si le 5173 est déjà utilisé).

---
